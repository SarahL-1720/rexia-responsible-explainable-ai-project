# REXAI Project

## Description

This project is part of the REXAI (Responsible Explainable AI) lecture at CentraleSupélec. The goal is to apply XAI principles on datasets of various domains to understand, explain and assess ethical implications of AI models.

## Usage

### Installation

We recommand using python 3.12 or higher with a virtual environment. You can install the required libraries using pip:

```bash
pip install -r requirements.txt
```
